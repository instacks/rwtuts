RSS Widget 2

Configure this stack by using the given Settings Display on the right. Please note that you need to provide a valid RSS Feed URL. These URLs usually have a .xml, .rss or /rss extension.

The RSS Widget 2 Stack is powered by FeedWind, a powerful RSS Feed provider.
